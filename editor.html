Level Editor

← back to the game. Draw / Erase terrain with the mouse (click or drag), or import an image (dark pixels become solid ground). Use Place hatch / Place exit then click the map to position them. Decor stamps a scenery prop on the surface (click a prop to remove it); Pillar drags out a region that the game paints as a fluted column (click one to remove it). Set the parameters and export the same levels.json the game loads.
